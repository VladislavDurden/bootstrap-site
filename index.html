When I say ETU, I mean "easy to understand". So on this web-site you can find some usefull information about programming.

First of all you should choose your own way.

Reasons can be very different:

  • Somebody have mates-programmers in some area, so in difficult situation they can ask for support.

  • You may learn programming in school/university and enjoy this language.

  • Or viewed film or TV series about programming fascinated you.

So, it's not important. I prepared for you some information which help you to understand main aspects of one or another direction of programming.

This area is focused on the development of Web applications (by another words, sites, but at the moment sites have such rich functionality that they can be called full-fledged applications).

Web-programming can be divided into backend (writing server scripts - PHP, Python, Ruby) and frontend (development of the user interface - Javascript, HTML, CSS).

Here you can see list of the most popular programming languages and it descriptions. Enjoy yourself.

  • JavaScript

    It's main front-end language. When you surf the Internet and see some animation or some things that change when you doing something on the web-site - it's JS !

  • Python

    This language also use in Web, but not only. Python using for improving developer productivity and code readability. Currently used for programming Neural Networks.

  • PHP

    It is one of the most popular back-end language. Usually using in not very massive projects. But Facebook and Wikipedia also using PHP.

  • Java

    Strong and so popular. Java using in Web like server language, applications for android are written on Java. RuneScape, Amazon, eBay, LinkedIn, Yahoo! using this language.

  • C#

    May be C# not so popular nowadays, but framework .NET is very popular. It's also back-end language and it is often compared with java

  • C++

    Widely used for software development, being one of the most popular programming languages. Using for creation of operating systems, device drivers and games

  • Swift

    Created by Apple primarily for iOS developers and macOS. It was created to replace the obsolete Objective-C.

  • Ruby

    Using for creation of simple and at the same time understandable programs,where the speed of the program is not important, but the short development time and simplicity

  • HTML

    HA-HA-HA, classic.

    Yes, of course it's not programming language, but a lot of people think so. HTML stands for HyperText Markup Language, so it's only frame of web-site page.

In this section you can read about three interesting things without which programming can not be programming.

  • API
  • Libraries
  • Frameworks

In plain language it's the ability for an application to access code outside this application.

So if it's difficult to understand, imagine the situation: you have a plumbing in the apartment, and the API - plumber's phone, if you need you call him and he repairs the water pipe.

It is a set of functions that someone once wrote so successfully that for some tasks this is the best choice.

And so as not to "CTRL+C CTRL+V" every time, this someone create single file which you cab include in your project.

These are software products that simplify the creation and support of technically difficult or loaded projects.

The framework dictates the rules for building the application architecture, setting the default behavior at the initial stage of development.
