When I say ETU, I mean "easy to understand". So on this web-site you can find some usefull information about programming.

  • JavaScript

    It's main front-end language. When you surf the Internet and see some animation or some things that change when you doing something on the web-site - it's JS !

  • Python

    This language also use in Web, but not only. Python using for improving developer productivity and code readability. Currently used for programming Neural Networks.

  • PHP

    It is one of the most popular back-end language. Usually using in not very massive projects. But Facebook and Wikipedia also using PHP.

  • Java

    Strong and so popular. Java using in Web like server language, applications for android are written on Java. RuneScape, Amazon, eBay, LinkedIn, Yahoo! using this language.

  • C#

    May be C# not so popular nowadays, but framework .NET is very popular. It's also back-end language and it is often compared with java

  • C++

    Widely used for software development, being one of the most popular programming languages. Using for creation of operating systems, device drivers and games

  • Swift

    Created by Apple primarily for iOS developers and macOS. It was created to replace the obsolete Objective-C.

  • Ruby

    Using for creation of simple and at the same time understandable programs,where the speed of the program is not important, but the short development time and simplicity

  • HTML

    HA-HA-HA, classic.

    Yes, of course it's not programming language, but a lot of people think so. HTML stands for HyperText Markup Language, so it's only frame of web-site page.
